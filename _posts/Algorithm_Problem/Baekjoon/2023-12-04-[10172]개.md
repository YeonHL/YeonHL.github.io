문제 링크: https://www.acmicpc.net/problem/10172

- Escape sequence를 조심해서 출력해야 한다.

## 1. Python

```python
print("|\_/|")
print("|q p|   /}")
print("( 0 )\"\"\"\\")
print("|\"^\"`    |")
print("||_/=\\\\__|")
```

<br>

## 2. Java

```java
public class Main {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder();

        sb.append("|\\_/|\n");
        sb.append("|q p|   /}\n");
        sb.append("( 0 )\"\"\"\\\n");
        sb.append("|\"^\"`    |\n");
        sb.append("||_/=\\\\__|");

        System.out.println(sb);
    }
}
```
